# Reinforcement Learning

Reinforcement learning (RL) is an area of machine learning concerned with how intelligent agents ought
to take actions in an environment in order to maximize the notion of cumulative reward. Reinforcement
learning is one of three basic machine learning paradigms, alongside supervised learning and unsupervised learning.
